Hvorfor?

For at gøre projektet nemmere at udvikle og smartere at bruge angående funktioner og design,
så kan vi benytte os af et udvalg af værktøjer via node.js.

budo

Budo er et værktøj som gør det mulight for os at udføre "live-relode"
På denne måde behøver vi ikke at genindlæse browseren hver gang at der
foretaget en ændring i HTML, JS eller CSS filerne
dette gøres automatisk hver gang at der er foretaget en ændring og
udvikleren er stoppet med at taste.
Budo bliver installeret via "npm install budo -g"
og kan efterfølgende kobles til javascript filen via "budo index.js --live --open"

nodemon

Nodemon minder lidt om budo. Men i stedet for at genindlæse browseren.
Så genstarter nodemon node applikationen hver gang at der er foretaget ændringer.
Nodemon installeres med "npm install -g nodemon" og kører applikationen med
"nodemon [your node app]"

jQuery

jQuery er et javaScript bibliotek, som gør det nemmere for os at håndterer ting som HTML og CSS manipulation.
Dette kunne f.eks. være tilføjelse at bilelder, ændre farver, fjernelse af elementer og meget mere.
Da jQuery er et javaScript bibliotek så vil koden kunne findes i vores .js filer.

Postman

Som udgangspunkt understøtter browsere ikke post eller put metoder. Derfor er Postman blevet udviklet som giver os denne mulighed.
Via postman kan vi forbinde til serveren ligesom med en browser forbinde til serveren via localhost.
Her kan vi vælge imellem GET, POST, PUT DELETE mfl. Hvis vi her vælger POST, så skal vi under "body" vælge "raw" typen
og her efter JSON. På denne måde kan vi sende JSON objekter til serveren som herefter kan blive behandlet af vores javascript.
På billedet nedenfor kan vi se hvordan vi tilføjer elementer til vores program via Postman og POST.

Det er som sagt også muligt at ændre i vores liste via PUT. Dette kan ses nedenfor.
Ved at angive elementets id i url'en så kan vi bruge denne parameter til at få fat i den
enkelte car og ikke alle i vores collection. Herudover sender vi et JSON objekt med "brand" som
fortæller hvilket element i vores car vi ønsker at ændre samt den værdig vi ønsker at ændre til.

Koden for at opdaterer i listen kan ses nedenfor. Koden er kommenteret på engelsk.

Til sidst er muligheden for at slette elementer. Her bliver elementet slettet ud fra
den parameter som er givet i url'en og herefter bliver den nye list sendt som response.

Bulma

Bulma er et online css bibliotek som gør det muligt for os, super hurtigt at designe vores
HTMl sider med farver og orienteringer. Bulma fungerer på den måde at man linker til deres
onlinebibliotek "https://cdn.jsdelivr.net/npm/bulma@0.8.0/css/bulma.min.css" som man ville med
en normal css fil. Herefter tildeler man bare sine html tags med <class=""> og inde i
teksfeltet skriver man diverse classenavne som er definerert i biblioteket. f.eks. ville man
kunne skrive "<button class="button"> With bulma </button>"" og det ville automatisk designe knappen.

Det er muligt at tilegne flere bulma klasser til et tag. Dette gøres ved et mellemrum imellem
klassenvnene. På denne måde ville vi f.eks. kunne ændre størrelse og farve på den allerede designede knap.
Følgende linje vil skabe nedestående knap button <class="button is-info is-small is-rounded">Rounded Button</button>
